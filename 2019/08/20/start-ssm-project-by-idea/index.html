使用idea创建 springMVC + MyBatis + jsp 的web项目
发布于：2019-08-20 16:41:31 标签：/ idea / spring / springmvc / jsp / 访问：

本文的项目地址：SSMWebDemo

1.打开idea，新建项目

2.左侧选中maven，勾选 Create from archetype，在列表中找到 maven-archetype-webapp

3.填写 GroupId 和 ArtifactId

4.填写maven配置，主要设置 User setting file (本地maven配置文件) 和 Local respository (本地仓库位置)

5.填写项目名称和项目路径

6.进入编辑模式，如果弹出 Maven projects need to be imported，选择 Enable Auto-import即可，以后更改pom文件后就会自动下载依赖包了

7.将生成的pom文件中的 build 内容删除，之后按需求自己配置，目前为止整个项目的目录结构是这样的

8.点击菜单栏上的 File-> Project Structure-> Project Settings-> Modules，在 src/main 下点击右键-> New Folder，新建两个文件夹:java 和 resources

9.将新建的java文件夹设置成Sources ,新建的 resources文件夹设置成Resources，具体操作：点击java文件夹，点击上面的Sources 按钮。

10.在pom文件中添加我们需要的依赖，主要包括spring核心依赖，spingMVC依赖，mybatis的相关依赖，jsp的依赖，servlet的依赖，数据库连接池(本文选用druid)的依赖，为了简化操作，首先在中指定spring的版本，主要配置如下

11.在resources文件夹下创建配置文件：applicationContext.xml，applicationContext-beans.xml，applicationContext-jdbc.xml，applicationContext-mybatis.xml，spring-mvc.xml，jdbc.properties，mybatis.cfg.xml。具体代码如下：

  • applicationContext.xml

  • applicationContext-beans.xml

  • applicationContext-jdbc.xml

  • applicationContext-mybatis.xml
  • mybatis.cfg.xml
  • jdbc.properties

12.修改web.xml文件。

13. 在webapp文件夹下新建jsp文件夹，将原本生成的 index.jsp 移动到jsp文件夹下。

  • index.jsp

14. idea右上角菜单栏->Edit Configuration -> 点击 + 号添加tomcat

15. 点击Application server 后面的 Configure按钮 配置指向本地tomcat路径

16. 点击Deployment -> + 号 -> 你的项目的war包名称，例如demoWeb.war，修改下面的 Application context

17. 在java目录下新建com/aias/admin/,建四个子包 action/bean/dao/service，在action中新建DemoController

18. 点击run/debug启动tomcat

19. 启动成功会自动弹出一个浏览器界面，则说明配置成功

20.新建数据库表t_user

21.修改相关代码如下：

  • com/aias/admin/bean/User.java
  • com/aias/admin/dao/IUserDao
  • com/aias/admin/bean/UserMapper.xml
  • com/aias/admin/service/IUserService.java
  • com/aias/admin/service/impl/UserServiceImpl.java
  • com/aias/admin/action/DemoController.java
  • webapp/jsp/index.jsp

23.在pom文件中添加打包mapper的插件

22.启动tomcat，启动完成之后访问http://localhost:8080/demo/list

配置完成
