使用idea创建SpringBoot项目

1.打开idea-> Create New Project

2.选择Spring Initalizr

3.填入自定义的Group和Artifact，其他内容根据自己需求更改，这里均使用默认值

4.选择合适的SpringBoot 和需要的依赖，这里选择的依赖有Lombok、SpringBootDevTools，因为要创建web项目，所以还选择了webStarter

5.选择项目在本地保存的位置和项目名称

现在项目已经创建成功，项目开始初始化，接下来开始进行配置

1.修改pom.xml，增加自定义依赖，这里主要添加了jdbc、mybatis、druid数据库连接池以及fastjson的依赖，并添加了编译打包的插件：

将代码项目设置为模板工程
