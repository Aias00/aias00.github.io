本篇不写参与试用方法,只记录启动实例之后做的一系列操作,只要是 ubuntu-18.04 版本的系统都可以参考此操作流程

更新 apt

apt-get update

安装 shadowsocks-libev

因为服务器在东京,正好手头没有合适的梯子,不利用起来就是浪费

2.安装 git

apt install git -y

3.获取源码并编译

首先要进入到你想要保存源码的目录,比如/usr/local/xxx(自己建的文件夹)

4.创建配置文件

配置文件内容,记得修改密码

5.创建 Shadowsocks-libev.service 配置文件,用于配置服务启动

6.启动服务
